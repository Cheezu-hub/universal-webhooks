# Contributing to Universal Webhook Adapter

First off, thank you for considering contributing to the Universal Webhook Adapter! It's people like you that make the open source community such an amazing place to learn, inspire, and create.

## 🚀 How to Contribute

### 1. Reporting Bugs
- Check the [Issues](https://github.com/Cheezu-hub/universal-webhooks/issues) to see if the bug has already been reported.
- If not, create a new issue. Provide a clear description and steps to reproduce.

### 2. Suggesting Enhancements
- Open a [Feature Request](https://github.com/Cheezu-hub/universal-webhooks/issues/new) to discuss your idea.

### 3. Pull Requests
1. Fork the repo and create your branch from `main`.
2. If you've added code that should be tested, add tests.
3. Ensure the test suite passes.
4. Make sure your code follows the style guidelines (Prettier for JS/CSS, PEP8 for Python).
5. Submit a pull request!

## 📄 Style Guidelines

### Python
- Follow [PEP 8](https://www.python.org/dev/peps/pep-0008/).
- Use descriptive variable names.
- Document functions and classes with docstrings.

### Javascript (Frontend)
- Use **Prettier** for formatting.
- Follow the component structure in `src/components`.
